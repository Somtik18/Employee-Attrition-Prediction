
Employee Attrition Prediction and HR Analytics Dashboard

Project Overview

This project develops a machine learning-based employee attrition prediction system and HR analytics dashboard using the IBM HR Analytics Employee Attrition dataset.

The objective is to identify factors influencing employee turnover and support workforce retention strategies through predictive analytics and data visualization.

Data set

Source: IBM HR Analytics Employee Attrition Data set

Records: 1470 Employees

Features: 35 Employee Attributes

Target Variable: Attrition (Yes/No)

Project Objectives
 • Data Collection and Prepossessing
 • Exploratory Data Analysis (EDA)
 • Employee Attrition Analysis
 • Workforce Trend Analysis
 • Machine Learning Model Development
 • Model Evaluation and Comparison
 • Feature Importance Analysis
 • HR Analytics Dashboard Development

Technologies Used
 • R
 • RStudio
 • Shiny
 • Logistic Regression
 • Decision Tree
 • Random Forest

Exploratory Data Analysis Findings
 • Employees working overtime were more likely to leave the company.
 • Employees with lower monthly income had higher attrition rates.
 • Younger employees were more likely to leave the organization.
 • Employees living farther from work showed higher attrition.
 Machine Learning Results

Logistic Regression

Accuracy: 84.81%

Decision Tree

Accuracy: 83.67%

Random Forest

Accuracy: 82.77%

Best Performing Model

Logistic Regression achieved the highest accuracy of 84.81%.
